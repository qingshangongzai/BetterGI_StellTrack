# BetterGI 星轨（BetterGI StellTrack）
亲爱的旅行者，欢迎来到 BetterGI 星轨的项目主页

BetterGI 星轨是一款由 AI 大模型驱动、基于 Python 开发的创新软件，专为 BetterGI（一款《原神》自动化辅助工具）打造，致力于生成强大的键鼠自动化脚本。

通过其直观的可视化界面，您可以轻松构建复杂的自动化操作流程，从而显著提升游戏体验与效率。
